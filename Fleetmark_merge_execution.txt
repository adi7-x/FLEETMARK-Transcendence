CONTEXT — READ THIS FIRST
════════════════════════════════════════════════════════

This is a school shuttle reservation platform called Fleetmark
built for 1337 School, Ben Guerir, Morocco.

Students use it to reserve seats on night shuttles
that run from 10PM to 6AM.

THE TEAM:
  Adil Bourji       → Frontend (React + Vite + TypeScript)
  Mohamed Lahrech   → Backend (Django REST)
  Abderrahman Chakour → Backend / Auth (42 OAuth)
  Ayoub El Haouti   → Backend / QA
  Aamir Tahtah      → DevOps / Docker

THE STACK:
  Frontend: React 18 + Vite + TypeScript + Tailwind v4
  Backend:  Django REST Framework
  Auth:     42 Intra OAuth2 + JWT (SimpleJWT)
  DB:       PostgreSQL
  Docker:   docker-compose with 4 services

THE ROLES:
  STUDENT          → books shuttle seats
  LOGISTICS_STAFF  → admin, manages buses/routes/trips
  DRIVER           → drives (coming soon page for now)

API BASE URL: http://localhost:8000/api/v1

KEY FIELD NAMES (do not rename these ever):
  User:        id, login_42, email, role, station (uuid), station_name, is_active
  Station:     id, name
  Bus:         id, name, plate, seat_capacity
  Route:       id, name, window (peak|consolidated), stations[]
  Trip:        id, route, bus, driver, departure_datetime, seats, archived_at, status
  Reservation: id, trip, student, created_at
  Driver:      id, name, username, status

    2. If no code → navigate to /
    3. Call handleOAuthCallback(code) from services/api
